Crime in Washington DC

This is an experimental visualization, created using the MapBox API to display a beautiful and interactive map. D3 for JavaScript is used for visualizing the geoJSON data sets of crime and streetlights in Washington DC.

The functionality of the visualization is limited, thus making it purely experimental, however it is an interesting alternative to using a flat 2D map. The map can be explored closely, by zooming in with the scroll wheel, and by holding the left mouse button, it is possible to rotate and pitch the map, to see it from different angles. It also shows details such as area and street names, which can be very helpful for exploratory data analysis.

The datapoints are plotted on the map, using D3 as an overlay. Crime incidents are plotted as red dots on the map, and the streetlights are displayed in bright yellow. By hovering over the red dots, the tooltip shows which crime was committed and at what time of day. The crime data is only for 2017, as loading it from 2010-2017, would take too long.

Given more time, this could eventually have been used as the main visualization, as 3D hexbins could have been implemented to show crime or light density within the city.
